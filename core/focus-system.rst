Focus System
============

Méthodes
--------

- 25/5 (Pomodoro)
- Deep Work 60–90 min
- No multitâche

Anti-distractions
-----------------

- Téléphone hors de vue
- Notifications coupées
- Bureau rangé

Priorisation
------------

- Une seule priorité par jour
- 3 tâches maximum
- Revue rapide matin/soir
