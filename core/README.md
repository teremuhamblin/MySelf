# 🧩 Core — MySelf

Ce dossier contient les éléments fondamentaux du système MySelf : identité, règles et doctrine.

## 📁 Structure
- `Identity.rst` — définition de l’identité personnelle  
- `Rules.rst` — règles simples pour guider l’action  
- `Doctrine.rst` — principes directeurs et ligne de conduite  

## 🎯 Rôle
Fournir la base conceptuelle du système MySelf : qui je suis, comment j’agis, et selon quels principes.

## 📝 Format
Chaque fichier reste court, clair et orienté vers l’essentiel.
