# 🔐 Politique de Sécurité — MySelf

## 📣 Signalement d’une vulnérabilité
Si tu découvres une faille de sécurité, merci de la signaler via :

👉 https://github.com/TheMadDoG/MySelf/security/advisories

Ne publie **jamais** publiquement une vulnérabilité avant qu’elle ne soit corrigée.

## ⏱️ Délais de réponse
- Accusé de réception : 48h
- Analyse : 7 jours
- Correctif : selon criticité

## 🛡️ Versions supportées
| Version | Support |
|--------|---------|
| v1.x   | ✔️ |
| v0.x   | ⚠️ Correctifs critiques uniquement |
