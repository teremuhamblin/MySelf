✅ 1. Site web statique pour MySelf

Tu vas obtenir un mini‑site statique simple, propre, lisible, basé sur :

- HTML5 minimaliste  
- CSS moderne (variables, dark mode, responsive)  
- Structure modulaire  
- Pages générées depuis tes fichiers .rst (ou copiées manuellement)

📁 Arborescence du site

À placer dans :

```text
MySelf/website/
```

Structure :

```text
website/
│
├── index.html
├── style.css
├── script.js
├── pages/
│   ├── identity.html
│   ├── doctrine.html
│   ├── routines.html
│   ├── goals.html
│   ├── tracking.html
│   └── dashboard.html
└── assets/
    └── logo.svg
```
