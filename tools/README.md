 # 🛠️ Tools — MySelf

Ce dossier regroupe les outils utilisés par le système MySelf.

## 📁 Structure
- `templates/` — modèles réutilisables (fichiers, structures, formats)
- `scripts/` — scripts simples pour automatiser certaines actions

## 🎯 Rôle
Fournir des ressources pratiques pour faciliter la création, l’organisation et l’automatisation dans MySelf.

## 📝 Format
Chaque outil reste minimaliste, clair et orienté vers l’usage direct.
