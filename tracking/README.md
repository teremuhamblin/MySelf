 # 📊 Tracking — MySelf

Ce dossier regroupe les journaux utilisés pour suivre la discipline, la santé et la progression.

## 📁 Structure
- `Discipline.log` — suivi des actions liées à la discipline  
- `Health.log` — suivi santé (sommeil, activité, alimentation)  
- `Progression.log` — suivi global de l’avancement  

## 🎯 Rôle
Fournir une trace simple et régulière pour mesurer l’évolution personnelle.

## 📝 Format
Chaque entrée suit un format court :  
`[DATE] — Action / Observation`
